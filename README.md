#Proyecto: Herramienta de producción
Esta herramienta se basa en 3 funciones:
- Producción
- Consumo
- Asistencia

Las bases son esas. En producción se conseguirá entender qué tenemos y qué hacemos con todo ello. Tenemos información, pero no la usamos. A partir de esta herramienta se usará y compilará todo aquí. Necesario para el forfait.

En consumo se conseguirá el coste Kg/L de la fábrica. Se entenderá el coste para el forfait del producto.

En asistencia se buscará entender la eficiencia de la fábrica según el número de trabajadores. También se mantendrá un control de las faltas y asistencias. Esto se usará para el forfait también.
